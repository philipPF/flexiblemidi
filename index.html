FlexiBLE MIDI is a webpage created in 2017 to help people connecting bluetooth midi equipement on their android device, and being able to send midi informations via USB.

It is based on :

  • MIDI Web API
  • Bluetooth BLE Web API

To use bluetooth on web, you need to :
- activate the enable-experimental-web-platform-features in their Chrome configuration (copy paste this adress : chrome://flags/#enable-experimental-web-platform-features )
- access this site with HTTPS

These API are EXPERIMENTAL.
(Tested on Android 7.1.1 / Chrome 62)
